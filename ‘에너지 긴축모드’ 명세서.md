# 📝 프로젝트: 에너지 긴축 모드 (Energy Essential Mode) 기능 명세서

## 1. 프로젝트 개요

- **목표:** 번아웃 방지를 위한 힐링 타이머 & 메타인지 대화 서비스
- **플랫폼:**  **모바일 최적화 웹 애플리케이션 (Mobile-First Web Application)**
- **핵심 컨셉:** 번아웃이 온 사용자가 하나씨에게 지금 바로 할 수 있는 가장 작은 할 일을 요청하고, 사용자의 에너지 레벨에서 감당 가능한 할 일을 하며 무기력을 탈출하는 동안, 캐릭터(하나 씨)도 바위를 가꾸며 함께 할 일을 하며 동기부여를 함.

```
// 초대 코드: 인증된 사용자만 읽기 가능
match /inviteCodes/{codeId} {
  allow read: if request.auth != null;
  allow write: if false; // 코드 생성은 서버에서만

match /users/{userId} {

  // 세션 서브컬렉션
  match /sessions/{sessionId} {
    allow read, write: if request.auth != null && request.auth.uid == userId;
